Hardware Integration Protocol

Hook up physical RFID readers instantly without deploying separate backends.

Method A: USB Keyboard Emulators (Recommended)

Most commercial USB RFID card readers act as passive standard keyboards (frequently called "Keyboard Wedges"). No programming necessary!

  1. Simply plug the USB cable of your RFID scanner reader into any computer USB port.
  2. The hardware scanner will light up, indicating operational readiness.
  3. Open the Live Scan Monitor tab in this app.
  4. Focus your pointer on the page. Swiping card characters directly mimics typed sequences ending with the "Enter" key, triggering system check-ins.

Method B: Web Serial API (Direct Microcontroller Hookups)

Connect microcontrollers directly (such as Arduino, ESP32, or Raspberry Pi) linked with RC522 or PN532 RFID readers!

Arduino Serial Code Template

Flash this instruction to your Arduino to broadcast scanned RFID tags over the USB Serial interface: 

void setup() {
  Serial.begin(9600); // Connect at 9600 baud
}
void loop() {
  // Read RFID Card UID...
  if (cardDetected) {
    Serial.println(cardUID); // Print string ending with \n
  }
}
